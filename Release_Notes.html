Purpose

The STM32DDRFW-UTIL is the firmware package for DDR configuration and test.

This firmware contains a minimal set of drivers based on STM32CubeMP1xx Firmware package:

  • HAL drivers
  • CMSIS Device
  • Cortex-A CMSIS Core
  • BSP for STM32MP1 Series MPUs
  • imageheader

The DDR_Tool directory includes STM32CubeIDE Projects for all ST supported boards
